整体布局风格 — 你的个人岛屿

网站的核心隐喻是「一座属于你的岛屿」，访客来探索你的技能世界。哪种布局最吸引你？

A

岛屿地图式导航

主页是一张 SVG 岛屿地图，不同区域（家=关于、博物馆=博客、工坊=技能、海滩=放松）分布在地图上。点击区域进入对应页面。像真正的动森一样自由探索。

优点

  • 沉浸感最强
  • 探索趣味性高
  • 完美契合动森主题

缺点

  • 移动端适配较难
  • 信息查找效率较低
B

滚动式岛屿故事

单页滚动设计，从上到下依次经过：天空→树屋(技能)→博物馆(博客)→广场(关于)→海滩(放松)。每个区域有独立的动森风格场景和微交互。顶部有简洁的「快速旅行」气泡导航。

优点

  • 叙事流畅，引导性强
  • 移动端友好
  • 内容组织清晰

缺点

  • 页面可能较长
  • 自由度不如地图式
C

口袋式侧边导航

左侧一个类似「Nook Phone」的侧边栏，包含所有功能的图标按钮。右侧是主要内容区，背景根据当前页面变化（如博客=博物馆墙纸，技能=工坊木纹）。保留现有的侧边栏结构但全面升级。

优点

  • 信息架构清晰
  • 实现成本适中
  • 复用现有路由结构

缺点

  • 相对传统
  • 惊喜感不如前两者
