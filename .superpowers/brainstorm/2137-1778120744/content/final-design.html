🍃 开场：落叶登岛

  • 页面加载 → 暗绿背景 → 一片 SVG 大叶子从左上飘旋而入
  • 叶子落于画面中央 → 「化开」扩散为岛屿地图
  • 角色图标弹跳落在岛上 → 随机欢迎语气泡：「哩哈！欢迎来到我的岛～」
  • 二次访问（localStorage记录）→ 跳过动画直接显示

🎬 动效系统（加强版）

  • 全局氛围：缓缓飘动的云朵、偶尔飞过的蝴蝶 SVG、微风吹拂的草浪
  • 大树：叶片摇摆 + 随机落叶（CSS animation + JS interval）
  • 博物馆：萤火虫光点漂浮（SVG circle 动画）
  • 工坊：齿轮缓慢旋转 + 卡片翻转 3D 效果
  • 沙滩：海浪波纹来回拍打（CSS wave animation）
  • 小屋：烟囱炊烟袅袅 + 窗户暖光呼吸
  • 小径：页面切换时「走过小径」过渡，角色沿路径移动到目标
  • 悬停：建筑弹跳、卡片微微浮起、按钮膨胀
  • 彩蛋：点击特定位置出现摇晃的树、挖出化石的惊喜
