技能树展示 — 工坊里的「创造」

技能=你在岛上能「制作」的东西。如何展示这些能力，既专业又有趣？

✨ C/C++ ──┐ ├── STM32 ✨ C# ────┤ ├── NRF52 ✨ Python ┘

星座连线图

技能像夜空中的星座，用 SVG 线条连接相关的技术点。悬停时星座亮起，显示熟练度和简介。整体像一张「星图」——低调、神秘、高级。

亮点

  • 视觉冲击力强
  • SVG绘制自然优雅
  • 「星图」隐喻知识宇宙
🔌 嵌入式
📡 蓝牙
⚡ RTOS
🖥️ C# GUI
🐍 Python
🌐 Web

制作台卡片墙

技能以「制作配方卡片」形式展示。每张卡片是一个技术领域，翻转后显示详情。动森的 DIY 工作台概念——「我能用这些材料制作东西」。

亮点

  • 最贴合动森工坊设定
  • 卡片翻转交互有趣
  • 分类清晰直观

博物馆展柜

技能像博物馆的展品陈列在玻璃展柜中，按类别分区（化石翼=嵌入式、鱼类=C#应用、昆虫=Web技术）。每个展柜点击后有详细解说牌。博物馆的庄严感=「高大上」。

亮点

  • 高大上的博物感
  • 分类陈列有仪式感
  • 解说牌文字可以很幽默
